## 1. 제품 개요

### 1.1 비전
"개인 투자자의 전문 애널리스트" — 기관 수준의 심층 분석을 누구나 쉽게 접근할 수 있게 하는 AI 주식 분석 웹 서비스

### 1.2 한 줄 정의
AI가 전문 애널리스트 수준으로 종목을 분석해주고, 일반인이 놓치는 숨겨진 인사이트를 제공하는 B2C 주식 컨설팅 플랫폼

### 1.3 핵심 철학
- **깊이 우선**: 매수/매도 시그널보다 "왜"를 설명하는 전문 분석
- **접근성**: 금융 문외한도 이해할 수 있는 언어로
- **지속성**: 사용자가 계속 돌아오는 고품질 인사이트

## 2. 타겟 사용자

### 2.1 주요 페르소나
| 페르소나 | 설명 |
|---|---|
| **호기심 투자자** | 주식은 하는데 분석은 어려운 직장인 (30-40대) |
| **정보 수집가** | 다양한 소스의 정보를 한 곳에서 보고 싶은 사람 |
| **포트폴리오 관리자** | 내 보유 종목의 건강 상태를 알고 싶은 사람 |

### 2.2 사용자 Pain Point
- 전문 리포트는 비싸거나 접근 어려움 (증권사 HTS, 유료 서비스)
- 뉴스 홍수 속에서 진짜 중요한 정보 선별 불가
- 매수/매도 의견은 넘쳐나지만 근거가 불투명
- 내 포트폴리오 전체를 한 눈에 분석해주는 곳 없음

## 3. 핵심 기능 (MVP)

### 3.1 Feature 1: AI 종목 심층 분석
사용자가 종목 검색 시 다음 분석을 AI가 자동 생성:

**기술적 분석 (Technical)**
- RSI, MACD, 볼린저 밴드 종합 해석
- 지지선/저항선 AI 판단
- 거래량 이상 패턴 감지

**펀더멘털 분석 (Fundamental)**
- PER/PBR/EPS 동종 업계 비교
- 매출/영업이익 성장 트렌드 분석
- DCF 밸류에이션 기반 적정 주가 추정
- 부채비율, 현금흐름 건전성 평가

**숨겨진 인사이트 (Hidden Signals)**
- 기관/외국인 순매수 동향 분석
- 공매도 비율 추이 해석
- 내부자 거래(임원 매매) 패턴
- 옵션 시장 Put/Call Ratio 해석
- 연기금/스마트머니 동향

**센티먼트 분석 (Sentiment)**
- 뉴스/공시 자동 수집 + AI 요약
- 호재/악재 자동 분류 및 영향도 평가
- SNS/커뮤니티 분위기 지수

### 3.2 Feature 2: 포트폴리오 X-Ray
- 보유 종목 입력 → 전체 포트폴리오 리스크 진단
- 섹터 쏠림 분석, 상관관계 분석
- "이 조합의 약점은?" AI 조언

### 3.3 Feature 3: 실시간 알림 시스템
- 관심 종목 이상 신호 알림 (기관 대량 매수, 공시 발생 등)
- 주요 경제 지표 발표 전 사전 영향 분석

## 4. 수익 모델

### 4.1 광고 기반 Freemium
| 구분 | 무료 | 광고 시청 후 | 유료 (향후) |
|---|---|---|---|
| 종목 분석 | 기본 3개 항목 | 전체 분석 (광고 1회) | 무제한 |
| 포트폴리오 X-Ray | 3종목 제한 | 10종목 (광고 1회) | 무제한 |
| 알림 | 1개 종목 | 5개 종목 (광고 1회) | 무제한 |

### 4.2 광고 전략
- Google AdSense 기반
- 인사이트 "잠금 해제" 전 자연스러운 광고 노출
- 강제적이지 않은 UX (건너뛰기 가능, 단 기능 제한)

## 5. 기술 스택

### 5.1 확정 스택
| 레이어 | 기술 | 선택 이유 |
|---|---|---|
| **Frontend** | Next.js 14 + TypeScript | SEO, SSR, 생태계 |
| **UI** | Tailwind CSS + shadcn/ui | 빠른 개발, 고품질 컴포넌트 |
| **차트** | TradingView Lightweight Charts | 금융 차트 표준 |
| **Backend** | Python + FastAPI | AI/ML 통합 용이 |
| **AI** | Claude API (Anthropic) | 심층 분석 텍스트 생성 |
| **데이터** | yfinance + Alpha Vantage + OpenDART | 주가/재무/공시 |
| **DB** | PostgreSQL + Redis | 관계형 + 캐싱 |
| **인프라** | Vercel (FE) + Railway (BE) | 빠른 배포, 저비용 시작 |
| **모니터링** | Sentry + PostHog | 에러 + 사용자 행동 |

## 6. MVP 범위 (Phase 1)

### 포함
- [ ] 종목 검색 + 기본 AI 분석 (기술적 + 펀더멘털)
- [ ] 숨겨진 인사이트 2-3가지 (기관동향, 공매도, 공시 요약)
- [ ] 광고 잠금해제 UX (Google AdSense)
- [ ] 반응형 웹 (모바일 우선)
- [ ] 사용자 관심 종목 저장 (로컬스토리지, 회원가입 없이)

### 제외 (Phase 2+)
- 포트폴리오 X-Ray
- 실시간 알림
- 회원가입/로그인
- 유료 구독
